# portfolio

Playful, scroll-driven personal site. Next.js (App Router) + React.

## Run locally
Requires Node 20+.

    npm install
    npm run dev

Open http://localhost:3000

## Scripts
- `npm run dev` — local dev server
- `npm run build` — production build
- `npm run lint` — eslint
- `npm start` — serve the production build

## Deploy
Auto-deploys on Vercel when `main` is updated.
