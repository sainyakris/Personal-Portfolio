# Portfolio

A modern personal portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Three.js to showcase projects, technical writing, and work in AI engineering, creative technology, and full-stack development.

The portfolio is designed to provide a recruiter-friendly experience while highlighting real-world projects, technical skills, and insights from building intelligent systems.

## Features

### Interactive Hero Section

A visually engaging landing experience featuring smooth animations and interactive 3D elements powered by React Three Fiber and Three.js.

### Project Showcase

A curated collection of projects including:

* Agentic AI Project Manager
* Automatic Traffic Flow Optimizer
* WAP Book Marketplace
* Personal Portfolio

### Technical Blog

An MDX-powered blog with:

* Dynamic routing
* Syntax highlighting
* Reading time estimation
* Table of contents
* SEO optimization

### Responsive Design

Fully optimized for desktop, tablet, and mobile devices.

### Performance & Accessibility

Built with modern web standards, accessibility best practices, and performance optimization in mind.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev

# or

yarn dev

# or

pnpm dev

# or

bun dev
```

Open http://localhost:3000 in your browser to view the website.

The application automatically reloads when changes are made during development.

## Deployment

This project is fully compatible with Vercel deployment.

To deploy:

```bash
npm run build
```

You can then deploy directly through Vercel or connect the repository for automatic deployments.
